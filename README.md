# 前端开发学习之旅

> 25 个模块 · 8 个学习阶段 · 13 种可视化组件 · 400+ 知识点的交互式前端学习平台。
> 从 HTML 基础到面试冲刺，系统化覆盖前端工程师的完整知识体系。

这不是一份教程文档，而是一个**可运行、可交互**的学习项目：每个知识点都配可视化演示、代码沙盒、小测验或实战项目，让你在"动手做"中理解原理，而非死记结论。

---

## 一、为什么用它学

- **看见原理**：盒模型、事件循环、Fiber 工作循环、HTTP 握手……都用可视化组件动态呈现，而不是静态截图。
- **以测促学**：每个模块末尾有小测验（含梯度难度标注）和面试题，学了立刻自检。
- **动手实战**：关键模块配带断言检查的代码沙盒，写错即时反馈，写对立刻通过。
- **路径清晰**：25 个模块按 8 个阶段循序渐进，从零基础到面试冲刺一条线打通。

## 三、如何学习

### 在线浏览

启动开发服务器后，在浏览器打开：

- `/` — 首页：项目概览与学习路径可视化
- `/modules` — 全部模块列表（按阶段分组）
- `/modules/:slug` — 模块详情页（如 `/modules/css-fundamentals`）
- `/about` — 关于

### 单个模块的结构

每个模块由若干**知识点（KP）**组成，典型结构如下：

```
模块
├── 知识点 1~N   核心讲解 + 可视化演示
├── 知识点       综合实战项目（带断言检查的代码沙盒）
├── 知识点       面试题（手风琴折叠，含场景题/对比题）
├── 知识点       知识点速查表（浓缩记忆）
└── 知识点       小测验（梯度难度：记忆/理解/应用/对比/场景/综合）
```

- **可视化演示**：13 种交互组件，如盒模型可视化、事件循环模拟器、Fiber 工作循环、TCP 握手等。
- **代码沙盒**：实战项目配 `checks` 断言，实时反馈你的代码是否达标；自由探索型沙盒则鼓励你改参数观察变化。
- **小测验 + 面试题**：学完即测，题干标注难度梯度，每题带 `explanation` 解释"为什么"。

## 四、快速开始

### 环境要求

- Node.js（建议 LTS 版本）
- npm（或 pnpm / yarn）

### 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器（默认 http://localhost:5173）
npm run dev
```

### 常用脚本

| 命令 | 作用 |
| --- | --- |
| `npm run dev` | 启动 Vite 开发服务器（HMR 热更新） |
| `npm run build` | 类型检查 + 生产构建 |
| `npm run preview` | 预览生产构建产物 |
| `npm run lint` | ESLint 代码检查 |
| `npm test` | 运行全部单元测试（Vitest） |
| `npm run test:watch` | 监听模式运行测试 |
| `npm run test:coverage` | 生成测试覆盖率报告 |

## 八、给学习者的建议

1. **按编号顺序学**：模块编号即推荐学习顺序，基础知识会反复用在后续模块。
2. **动手改沙盒**：看到代码沙盒就改几个参数，观察可视化变化，比读三遍文档管用。
3. **做完小测验再走**：每个模块末尾的测验和面试题是检验掌握程度的标尺，不要跳过。
4. **卡住了看速查表**：每个模块都有知识点速查表，适合复习和查漏补缺。
5. **实战项目要达标**：带断言检查的沙盒会告诉你是否写对，全部通过再进入下一模块。
